Covid Cases

The first Map visualisation visualises the total of covid cases per kapita in each states of Australia.
The second Map visualisation visualises the total of covid cases per vaccination rolled out in each states of Australia.

The first line graph below shows the confirmed cases per day in each states.
You can filter which state you would like to observe by clicking on the state in the legend.
The area graph shows the number of vaccines rolled out to date in each states.
You can zoom in on a specific time range by selecting on the first line graph
The area graph and the second line graph will be updated according to the selected zoomed in time line.
